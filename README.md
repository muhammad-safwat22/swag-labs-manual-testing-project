# Swag Labs Manual Testing Project

## Overview
This repository presents a Manual Testing Project for the Swag Labs website. It demonstrates the end-to-end software testing process, including requirement analysis, test scenario design, test case writing, bug reporting, and test summary reporting.

## Tested Pages
- Login Page
- Products Page
- Cart Page
- Checkout Information Page
- Checkout Overview Page
- Checkout Complete Page

## Testing Activities
- Requirement Analysis
- Writing Test Scenarios
- Writing Test Cases
- Bug Reporting
- Test Summary Report

## Tools Used
- Microsoft Excel
- Google Sheets
